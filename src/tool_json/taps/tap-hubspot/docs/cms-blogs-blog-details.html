Blog details

This is a stable API but uses an older format. This means some functionality may not be available (for example test calls and Postman integration).

Introduction

Use the blogs API to retrieve information about the blogs in a HubSpot account.

Use cases for this API

Review revision history for one of the blogs in your HubSpot account to understand when certain changes were made.

List blogs

GET/content/api/v2/blogs

List all of the blogs for an account. Supports paging and filtering.

If a parameter supports a filter type of 'gt', that means you can append a double underscore to the parameter name to find all items greater than the specified value. For example, the following query will find all blogs created after 5/1/2015: https://api.hubapi.com/content/api/v2/blogs?created__gt=1430452800000

Requirements
Limits
Standard API rate limits
Auth Methods
Private appsOAuth
Parameters
nameQuery param

The number of items to return. Defaults to 20

createdQuery param

The id of a topic you wish to filter blog posts by. Topic IDs can be found using the topics endpoint

archived_atQuery param

The offset set to start returning rows from. Defaults to 0.

offsetQuery param

exact, gt, gte - When the post was deleted, in milliseconds since the epoch. Zero if the blog post was never deleted. Use a DELETE request to delete the post, do not set this directly

topic_idQuery param

exact, range, gt, gte, lt, lte - When the post was first created, in milliseconds since the epoch

limitQuery param

exact, in - The internal name of the blog
