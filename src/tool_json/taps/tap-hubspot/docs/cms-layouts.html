Layouts

This is a stable API but uses an older format. This means some functionality may not be available (for example test calls and Postman integration).

Introduction

Use the layouts API to retrieve data about global groups for CMS templates in the developer file system. These layouts can be found in the design manager.

Use cases for this API

Retrieve version history for a global group to understand when certain changes were made.

List layouts

GET/content/api/v2/layouts

Get all layouts for an account. Supports paging and filtering.

If a parameter supports a filter type of 'gt', that means you can append a double underscore to the parameter name to find all items greater than the value. For example, the following query will find all items created than the 'updated' time:

Scopes

Limits

Auth Methods

Parameters
limitQuery param

The number of items to return. Defaults to 20.

offsetQuery param

The offset set to start returning rows from. Defaults to 0.

category_idQuery param

exact

createdQuery param

exact, range, gt, gte, lt, lte

deleted_atQuery param

Use a DELETE request to delete the layout - do not set this directly.

idQuery param

exact, in

labelQuery param

exact

pathQuery param

exact

custom_headQuery param

Custom HTML added to the head of pages using the layout.

include_default_custom_cssQuery param

Flag to include the Primary CSS stylesheet. Defaults to use value from Site Settings. Can be overridden at the page level.

enable_domain_stylesheetQuery param

Flag to include the domain stylesheets for a page using the layout. Can be overridden at the page level.

attached_stylesheetsQuery param

List of references to stylesheets that are attached directly to the layout. Can be overridden at the page level.

Gets the current contents of the auto-save buffer

GET/content/api/v2/layouts/:layout_id/buffer

The auto-save buffer is updated continuously when a user is editing the content through the UI. Only when the user clicks "Save" or "Publish" does the auto-save buffer get copied to the live object.
