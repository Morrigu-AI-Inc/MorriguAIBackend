Deal Splits

Introduction

Use the deal splits API to split existing deals between owners when credit and/or commission are shared between them.

Use cases for this API

When two sales reps work together to close a deal, add a split to the deal to reflect that the credit and commission are being shared.

Related guide:

Deal splits overview

Supported products

Requires one of the following products or higher.
  • Sales Hub - Enterprise

Batch

Create or replace deal splits for deals with the provided IDs. Deal split percentages for each deal must sum up to 1.0 (100%) and may have up to 8 decimal places
