Custom Activities

Custom Activities let you create arbitrary activity types with user defined fields.

To work with Custom Activities, you will need to:

  1. Create a Custom Activity Type.
  2. Add one or more Activity Custom Fields to that type or associate a Shared Custom Field with it.
  3. Once the Custom Activity Type is defined, create and manage Custom Activity instances of that type as you would any other activity.
