    Usage info

    This method allows searching accross all workflows in the top-level team of the caller. When sorting by premium runs, the results are not guaranteed to be in an exact ordering relative to the latest live data, since the run counts used for sorting may be out of sync by up to 24 hours. It returns an array of all workflows in the org, letting you filter by app ID, name, etc.

    This method requires the caller having the permission to manage apps for the workspace or enterprise.
