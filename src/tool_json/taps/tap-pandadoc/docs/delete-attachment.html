Delete Attachment

delete https://api.pandadoc.com/public/v1/documents//attachments/

Deletes an attachment from the document.

🚧

Can't delete evidence attachment

You'll get a 403 Forbidden error if you try to delete an attachment of the evidence type (proof of the manual status change).
